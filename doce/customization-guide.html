Black Hole WareHouse - Customization Guide

Welcome to your new Black Hole WareHouse website! This guide will help you customize the site to your specific needs.

Site Structure

The website has a modular structure with the following components:

  • index.html - Homepage with sections for hero banner, features preview, showcase, media, social links, and FAQ
  • pages/ - Directory containing individual pages:
    • features.html - Detailed features page with card layout
    • showcase.html - Product showcase with specifications
    • media.html - Videos and images gallery
    • social.html - Social media links and community
    • faq.html - Frequently asked questions
  • css/styles.css - Main stylesheet with customizable variables
  • js/main.js - JavaScript for interactive elements
  • images/ - Directory for your images (currently contains placeholders)

Quick Customization Guide

1. Changing Colors

The site uses CSS variables for easy color customization. Open css/styles.css and look for the :root section at the top:

:root {
  /* Main colors - easily customizable */
  --bg-dark: #121212;
  --bg-darker: #0a0a0a;
  --bg-lighter: #1e1e1e;
  
  /* Accent colors - easily customizable */
  --accent-purple: #6a1b9a;
  --accent-red: #c62828;
  --accent-blue: #1565c0;
  
  /* Text colors */
  --text-primary: #f5f5f5;
  --text-secondary: #b0b0b0;
  
  /* ... other variables ... */
}

Simply change these color values to match your preferred color scheme.

Tip: You can use any color format (hex, rgb, hsl) for these variables. For example, #6a1b9a could be changed to rgb(106, 27, 154) or hsl(282, 68%, 35%).

2. Adding Your Images

Replace the placeholder images with your own by:

  1. Adding your images to the images/ directory
  2. Updating the image paths in the HTML files

For example, to replace the hero image on the homepage, find this line in index.html:

<img src="images/placeholder-hero.png" alt="Black Hole Device" class="hero-image">

And change it to:

<img src="images/your-hero-image.jpg" alt="Black Hole Device" class="hero-image">
Note: Make sure your images are optimized for web (compressed) to ensure fast loading times. Recommended formats are JPEG for photos and PNG for graphics with transparency.

3. Modifying Content

Each page is divided into clear sections that you can easily modify:

  • Update text by editing the content between HTML tags
  • Add or remove sections by copying existing section structures
  • Modify navigation links in the navbar section of each page

Example of a section structure:

<section>
    <div class="container">
        <h2 class="text-center">Section <span class="text-purple">Title</span></h2>
        <p class="text-center mb-3">Section description goes here.</p>
        
        <!-- Section content goes here -->
        
    </div>
</section>

4. Adding New Pages

To add a new page:

  1. Copy one of the existing HTML files in the pages/ directory
  2. Rename it to your desired page name (e.g., new-page.html)
  3. Update the title, content, and active navigation link
  4. Add a link to the new page in the navigation section of all pages

Advanced Customization

Layout Modifications

The site uses a combination of flexbox and grid layouts for responsive design. To modify layouts:

  • For card layouts, adjust the grid-template-columns property in .features-grid and .media-grid classes
  • For showcase items, modify the flex properties in .showcase-item, .showcase-image, and .showcase-content classes

Adding Custom JavaScript

To add custom JavaScript functionality:

  1. Open js/main.js
  2. Add your custom code within the DOMContentLoaded event listener
  3. For larger customizations, create a new JS file and link it in the HTML files

Responsive Design Adjustments

The site is already responsive, but you can fine-tune the mobile experience by modifying the media queries at the bottom of css/styles.css:

@media (max-width: 992px) {
  /* Tablet adjustments */
}

@media (max-width: 768px) {
  /* Mobile adjustments */
}
