Frequently Asked Questions

Find answers to common questions about the Black Hole V1.0 device.

General Questions

Basic information about Black Hole V1.0.

What is Black Hole V1.0?

Black Hole V1.0 is a dual-band WiFi security testing device designed for educational purposes. It combines powerful hardware (ESP32 + BW16 chipset) with versatile software to provide a comprehensive platform for learning about WiFi security principles and testing network security.

What can I do with Black Hole V1.0?

Black Hole V1.0 allows you to:

  • Scan and analyze WiFi networks (both 2.4GHz and 5GHz bands)
  • Test the security of your own networks
  • Learn about WiFi security principles and vulnerabilities
  • Conduct authorized security assessments
  • Experiment with network protocols and security mechanisms

All of these activities should only be performed on networks you own or have explicit permission to test.

Is Black Hole V1.0 legal to use?

Black Hole V1.0 is designed as an educational tool for security testing on networks you own or have explicit permission to test. Using it on networks without authorization may be illegal in your jurisdiction.

Always check local laws and regulations before use. In many countries, unauthorized access to computer networks, interception of network traffic, and disruption of network services are criminal offenses that may result in severe penalties.

What's included in the Black Hole V1.0 package?

The standard Black Hole V1.0 package includes:

  • Black Hole V1.0 main unit
  • 2× 3dBi dual-band SMA antennas
  • USB-C charging cable (1m)
  • Quick start guide
  • Sticker pack

Optional accessories are available separately, including high-gain antennas, protective cases, and replacement batteries.

Technical Questions

Detailed information about specifications and capabilities.

What are the technical specifications of Black Hole V1.0?

Black Hole V1.0 features:

  • Main Processor: ESP32 dual-core @ 240MHz
  • Co-processor: BW16 for enhanced wireless capabilities
  • RAM: 520KB SRAM
  • Flash Memory: 4MB
  • WiFi: 802.11 b/g/n/ac (2.4GHz + 5GHz)
  • Bluetooth: BLE 5.0
  • Battery: 1200mAh LiPo (replaceable)
  • Dimensions: 80mm × 50mm × 15mm (without antennas)
  • Weight: 60g (with battery)
How long does the battery last?

The 1200mAh LiPo battery provides approximately 4-6 hours of typical usage. Battery life varies depending on the specific operations being performed:

  • Passive scanning: Up to 6 hours
  • Active testing: 3-4 hours
  • Continuous packet injection: 2-3 hours

The device can also be powered via USB-C while in use, allowing for extended operation.

Can I upgrade the firmware?

Yes, Black Hole V1.0 supports firmware updates via USB or over-the-air (OTA). We regularly release firmware updates with new features, performance improvements, and security patches.

The firmware is open-source, and advanced users can modify it to add custom functionality. Detailed instructions for updating and customizing the firmware are available in our documentation.

What is the range of Black Hole V1.0?

With the included 3dBi antennas, Black Hole V1.0 has an effective range of approximately:

  • Indoor: 30-50 meters (depending on obstacles)
  • Outdoor line-of-sight: 100-150 meters

Range can be extended by using optional high-gain directional antennas, which can increase the effective distance to 300+ meters in optimal conditions.

Usage Questions

Information about using and getting the most out of your device.

How do I access the web interface?

To access the Black Hole V1.0 web interface:

  1. Power on your Black Hole device
  2. Connect to the "BlackHole-Setup" WiFi network from your computer or mobile device
  3. Open a web browser and navigate to http://192.168.4.1
  4. The default login credentials are username: admin and password: blackhole

We strongly recommend changing the default password after your first login.

Can I use Black Hole V1.0 with my mobile phone?

Yes, Black Hole V1.0 features a responsive web interface that works on mobile devices. Simply connect your phone to the Black Hole WiFi network and access the interface through your mobile browser.

We also offer a companion app for Android and iOS that provides additional features and a more optimized mobile experience. The app is available for download from our website.

How do I update the firmware?

To update the firmware on your Black Hole V1.0:

  1. Download the latest firmware from our website
  2. Access the web interface as described above
  3. Navigate to Settings > System > Firmware Update
  4. Click "Upload Firmware" and select the downloaded file
  5. Click "Update" and wait for the process to complete

Alternatively, you can update via USB using the Black Hole Flasher Tool available on our GitHub repository.

What should I do if my device stops responding?

If your Black Hole V1.0 becomes unresponsive:

  1. Press and hold the reset button for 5 seconds to perform a soft reset
  2. If that doesn't work, press and hold both the power and reset buttons for 10 seconds to perform a hard reset
  3. If the device still doesn't respond, try connecting it to a USB power source and repeat the above steps

If these steps don't resolve the issue, please contact our support team for assistance.

Troubleshooting Guide

Solutions to common issues you might encounter.

My device won't power on

If your Black Hole V1.0 won't power on:

  1. Ensure the battery is properly installed and charged
  2. Connect the device to a USB power source and try again
  3. Press and hold the power button for at least 3 seconds
  4. If the device still won't power on, try removing and reinstalling the battery

If none of these steps work, the battery may need replacement or there might be a hardware issue. Please contact our support team.

I can't connect to the Black Hole WiFi network

If you're having trouble connecting to the Black Hole WiFi network:

  1. Ensure the device is powered on (the power LED should be lit)
  2. Wait at least 30 seconds after powering on for the WiFi network to initialize
  3. Check if the WiFi LED is blinking, indicating the network is active
  4. Try restarting your computer or mobile device's WiFi
  5. Reset the Black Hole device and try again

By default, the network name is "BlackHole-Setup" with no password. This can be changed in the settings once you've connected.

The web interface isn't loading

If the web interface isn't loading:

  1. Verify you're connected to the Black Hole WiFi network
  2. Try accessing the interface at http://192.168.4.1 (not https)
  3. Clear your browser cache and try again
  4. Try using a different browser
  5. Reset the device and try again

If you've changed the device's IP address in the settings, make sure you're using the correct address.

Firmware update failed

If a firmware update fails:

  1. Ensure you're using the correct firmware file for your device model
  2. Check that the device has sufficient battery (at least 50%) or is connected to USB power
  3. Try downloading the firmware file again, as it may have been corrupted
  4. Use the USB update method instead of the web interface

If the device is in a boot loop or completely unresponsive after a failed update, you can use the recovery mode. Press and hold the reset button while connecting the device to USB power, then use the Black Hole Flasher Tool to restore the firmware.
